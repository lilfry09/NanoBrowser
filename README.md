# NanoBrowser

一个轻量级的跨平台桌面网页浏览器，基于 Python 和 PyQt6 (Chromium内核)。本项目由 AI Agent 自动开发完成！

## 特性

- **多标签页支持**：支持打开多个网页并在标签页间切换。
- **智能地址栏**：自动补全 `https://`，非网址内容自动调用默认搜索引擎（默认：Bing）。
- **浏览历史记录**：自动记录并可查看您的访问历史。
- **书签管理**：将喜爱的网页一键加入书签，支持快捷访问。

## 运行方法

确保您的环境中有 Python 3。

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行浏览器
python src/main.py
```
